# Cybersecurity Portfolio CMS

A full-stack MERN Cybersecurity Portfolio with a custom Admin CMS.

## Features

* Admin Authentication (JWT)
* Dynamic Profile Management
* Skills Management
* Projects Management
* Certificates Management
* Badges Management
* Resume Management
* Image Upload Support
* Responsive Design
* MongoDB Atlas Integration
* React + Vite Frontend
* Express + MongoDB Backend

## Tech Stack

### Frontend

* React
* Vite
* React Router DOM
* Axios
* Tailwind CSS
* React Hot Toast
* React Icons

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* bcryptjs
* Helmet
* Express Rate Limit

## Installation

### Clone Repository

git clone <repository-url>

### Backend

cd backend

npm install

Create .env file

npm run dev

### Frontend

cd frontend

npm install

npm run dev
